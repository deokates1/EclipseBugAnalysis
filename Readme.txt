1. to download raw data visit http://2008.msrconf.org/challenge/index.html and download  "eclipse-bugs--000001-213000.zip"
2. extract the files in the folder containing the data mining python script
3. after running there will be two csv file one being the data.csv used in the jupyter notebook for the analytics part 
4. another csv file is created named xmlParsed1.csv. 
5. It conatains the list of all XML file stored in eclipse-bugs folder and if the xml was successfully parsed the name of root is stored else the error is stored.
6. when the notebook is opened in jupyter note just running all the cells will give the required output.
